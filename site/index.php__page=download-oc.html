Requirements

To run OpenCaps, the following is required:

  • Apache 2
  • PHP 5.2.9 with Zip lib
  • MySql 5
  • Firefox or Safari with Quicktime Plugin

Other versions/configurations may work but have not yet been tested.

Read the installation documentation for details on installing. Note: the Zip library for PHP is necessary for exporting a project package.
